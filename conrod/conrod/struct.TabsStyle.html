The styling for Canvas Tabs.

Fields

maybe_layout

The direction in which the tabs will be laid out.

maybe_bar_width

The width of the tab bar. For horizontally laid out tabs, this is the height of the bar. For vertically laid out tabs, this is the width of the bar.

maybe_label_color

The color of the tabs' labels.

maybe_label_font_size

The font size for the tabs' labels.

canvas

Styling for each of the canvasses passed to the Canvas.
