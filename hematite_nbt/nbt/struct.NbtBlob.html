An object in the Named Binary Tag (NBT) file format.

This is essentially a map of names to NbtValues, with an optional top- level name of its own. It can be created in a similar way to a HashMap, or read from an io::Read source, and its binary representation can be written to an io::Write destination.

These read and write methods support both uncompressed and compressed (through Gzip or zlib compression) methods.

use nbt::{NbtBlob, NbtValue};

// Create a `NbtBlob` from key/value pairs.
let mut nbt = NbtBlob::new("".to_string());
nbt.insert("name".to_string(), "Herobrine").unwrap();
nbt.insert("health".to_string(), 100i8).unwrap();
nbt.insert("food".to_string(), 20.0f32).unwrap();

// Write a compressed binary representation to a byte array.
let mut dst = Vec::new();
nbt.write_zlib(&mut dst).unwrap();
