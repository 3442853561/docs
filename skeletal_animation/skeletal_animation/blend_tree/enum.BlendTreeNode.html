pub enum BlendTreeNode<T: Transform> {
    LerpNode(Box<BlendTreeNode<T>>, Box<BlendTreeNode<T>>, ParamId),
    AdditiveNode(Box<BlendTreeNode<T>>, Box<BlendTreeNode<T>>, ParamId),
    ClipNode(ClipInstance<T>),
}

Runtime representation of a blend tree.

Variants

LerpNode

Pose output is linear blend between the output of two child BlendTreeNodes, with blend factor according the paramater value for name ParamId

AdditiveNode

Pose output is additive blend between the output of two child BlendTreeNodes, with blend factor according the paramater value for name ParamId

ClipNode

Pose output is from an animation ClipInstance
